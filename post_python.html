파이썬으로 웹 크롤링하기

개발동기

파이썬 입문용으로 쉽게 할 수 있는 웹크롤링을 해봤다. 내가 자주 보는 구제쇼핑몰의 사이즈 250짜리 신발이 새 제품이 나올 때 마다 알림을 받고 싶어서 만들게 됬다.

개발환경 및 Library

개발환경

  • 개발환경 : MacOS El Capitan Laptop
  • 운영환경 : CentOS release 6.8
  • 언어 : Python3

라이브러리

  • Python pip

    • Beautiful Soup4
    • Telepot

  • Telegram Messenger

    • Telegram Bot

사용기술?

  • Linux cronTab

시나리오

  • 파이썬 코드 작성

    • 원하는 URL을 HTML로 파싱
    • 신상품이 있거나, 신상품이 있지만 품절인 상품이 없는 경우만 다음으로 진행
    • 상품 목록 중 신상품만 Filter
    • 신상품 중 Sold Out이 아닌 것만 Filter
    • 상품의 이미지와 URL을 Telepot을 통해 TeleBot에게 전송

  • 텔레그램 메신저에서 제공해주는 TeleBot이 나에게 메세지 전송

  • Linux Crontab으로 Python Code를 매일 12시 6시 예약.

느낀점

  • 언제다 다 그렇지만 말로는 쉬워도 해보면 막상 그렇지 않다
  • 코드를 짤 때에는 충분히 설계를 하자
